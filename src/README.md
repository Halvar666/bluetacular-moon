# Bluetacular Moon

Bluetacular Moon is a complete theme for Pale Moon.

Bluetacular Moon is a fork of the classic Bluetacular theme by Nicholas Baudean.

This fork ports and adapts the visual identity of the classic Bluetacular Firefox theme to modern Pale Moon. It uses Past Modern Revisited as the initial compatible Pale Moon theme skeleton and gradually replaces or adjusts the visual layer to match Bluetacular where practical.

## Status

Release: 1.0.0

## Compatibility

Target application:

- Pale Moon 29.3.0 to 34.*
